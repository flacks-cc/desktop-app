## Acerca de Flack's Cut & Connect

Aplicación de escritorio construida con Java y Swing para la gestión administrativa integral de barberías. Proporciona control de inventario, reportes de ventas y administración de personal.

## Funcionalidades

- Gestión administrativa
- Control de inventario
- Reportes de ventas
- Administración de personal

## Inicio rápido

### Requisitos previos

- JDK 21+
- Apache Ant
- NetBeans IDE (recomendado)

### Instalación

```bash
git clone https://github.com/flacks-cc/desktop-app.git
cd desktop-app
ant build
```
