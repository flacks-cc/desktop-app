## 🚀 Inicio rápido

### Requisitos previos

- JDK 21+
- Apache Ant
- NetBeans IDE (recomendado)

### Instalación

```bash
git clone https://github.com/flacks-cc/desktop-app.git
cd desktop-app
ant build
```
